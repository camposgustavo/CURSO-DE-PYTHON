# 🐍 Python 3 – Exercícios e Anotações  
### Curso em Vídeo | Gustavo Guanabara

Este repositório contém exercícios resolvidos e anotações desenvolvidos durante o estudo do Curso de Python 3, ministrado pelo professor Gustavo Guanabara, no canal do YouTube Curso em Video.

---

## Objetivo do repositório

- Praticar lógica de programação por meio de exercícios.
- Organizar anotações das aulas.
- Disponibilizar um material de estudo e consulta futura.

---

O curso é dividido em três módulos , abordando desde os conceitos básicos até estruturas mais avançadas da linguagem.

---

#  Conteúdos Estudados 

## Mundo 1 – Fundamentos

-  Introdução ao Python.
-  Variáveis e tipos de dados.
-  Entrada e saída de dados.
-  Operadores aritméticos.
-  Manipulação básica de strings.

## Mundo 2 – Estruturas de controle
- Estruturas condicionais (if, elif, else).
- Laços de repetição (for, while).
- Controle de fluxo (break, continue).

## Mundo 3 – Estruturas compostas e funções

- Listas, tuplas e dicionários.
- Funções e parâmetros.
- Modularização.
- Tratamento de erros e exceções.

# Tecnologias utilizadas
- Python 3.14
- Visual Studio Code (VS Code)
- Git e GitHub
